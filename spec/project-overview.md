# Chess Analysis App — Огляд проєкту

## Короткий опис

Chess Analysis App — це вебзастосунок для імпорту шахових партій з Chess.com та Lichess, перегляду їх через Stockfish-подібний game review і генерації практичних LLM-порад для гравця рівня приблизно 1000 ELO.

Проєкт має реалізовуватись маленькими завершеними фазами. Кожна фаза повинна залишати застосунок у робочому стані.

## Ключові рішення v1

- Застосунок auth-first: користувач має увійти перед використанням продукту.
- Авторизація для старту: GitHub OAuth через NextAuth.js v5.
- NextAuth.js v5 використовує JWT strategy; `auth_sessions` не входить у v1-схему БД.
- Анонімний імпорт і анонімний аналіз не входять у v1.
- У v1 підтримуються обидві платформи: Chess.com і Lichess.
- Імпорт обмежений вибором користувача:
  - кількість партій: 25 / 50 / 100;
  - період: останні 7 / 30 / 90 днів.
- Значення за замовчуванням: 25 партій і останні 7 днів.
- У v1 застосунок не імпортує всю історію партій гравця за весь час.
- LLM-відповідь генерується українською.
- LLM-відповідь має бути строгим JSON, а не довільним markdown.
- Stockfish engine-аналіз зберігається в базі даних.

## Цільовий користувач

Основний користувач — шахіст, який покращує гру на рівні приблизно 1000 ELO і хоче отримувати зрозумілий прямий фідбек:

- що пішло не так у партії;
- які моменти змінили оцінку позиції;
- які тактичні або стратегічні помилки повторюються;
- що тренувати далі.

Поради мають бути практичними й конкретними, без термінів рівня гросмейстера, якщо вони не пояснені просто.

## Основні функції

### Автентифікація

- Вхід через GitHub OAuth.
- Сесійна авторизація через NextAuth.js v5.
- Session strategy: JWT.
- Dashboard, імпорт, перегляд партій, LLM-аналіз і профіль захищені авторизацією.

### Імпорт партій

- Користувач вводить платформу і нікнейм.
- Підтримані платформи: Chess.com і Lichess.
- Користувач вибирає кількість партій і період.
- Імпортовані партії зберігаються і прив'язуються до авторизованого користувача.
- Повторний імпорт додає тільки нові партії.

### Список партій

- Пагінований список імпортованих партій.
- Фільтри:
  - платформа;
  - контроль часу;
  - результат;
  - діапазон дат.
- Кожен рядок показує суперника, результат, дебют, дату і статус аналізу.

### Перегляд однієї партії

- Інтерактивна шахова дошка.
- Навігація по ходах.
- Список ходів у PGN/SAN-нотації.
- Підсвічування останнього ходу.
- Stockfish-подібний review:
  - eval bar;
  - eval graph;
  - стрілка найкращого ходу;
  - ключові моменти;
  - accuracy;
  - класифікація ходів.

### LLM-аналіз однієї партії

- Ручна дія: "Аналізувати цю партію".
- Вхідні дані: PGN, колір гравця, результат, контроль часу і доступні ключові моменти Stockfish.
- Вихідний формат: український JSON із такими блоками:
  - загальна оцінка;
  - дебют;
  - мідлгейм;
  - ендшпіль;
  - критичні моменти;
  - рекомендації.
- Аналіз кешується в базі даних.

### Груповий LLM-аналіз

- Користувач вибирає 5-30 партій.
- LLM отримує стислі зведення партій, а не повні PGN.
- Вихідний формат: український JSON із такими блоками:
  - повторювані закономірності;
  - тактичні слабкості;
  - стратегічні слабкості;
  - оцінка дебютів;
  - пріоритизований план дій.

### Профіль гравця

- Доступний після мінімум 5 імпортованих або проаналізованих партій.
- Фільтри дашборду:
  - 25 / 50 / 100 партій;
  - 7 / 30 / 90 днів.
- Метрики:
  - перемоги / нічиї / поразки;
  - результативність за контролем часу;
  - результативність за кольором;
  - найчастіші дебюти і win rate.
- Опціональне накопичувальне LLM-зведення.

## Поза межами v1

- Дебютний провідник.
- Рекомендації задач.
- Соціальний шерінг.
- Порівняння з друзями.
- Нативний мобільний застосунок.
- Платні тарифи або підписки.
- Відстеження партій у реальному часі.
- Анонімний режим.
- Email/password auth, reset password, email verification.
- Імпорт усієї історії партій.
- Користувацьке налаштування engine-порогів.

3. Перегляд однієї партії.
4. Stockfish game review.
5. LLM-аналіз однієї партії.

## Відкриті продуктові питання

- Профіль має об'єднувати Chess.com і Lichess за замовчуванням, чи потрібен перемикач платформи?
- LLM-аналіз однієї партії має вимагати готовий Stockfish-аналіз, чи може запускатися тільки по PGN?
